# Draft Landing — Training Project

Учебный лендинг, созданный для практики вёрстки, адаптивного дизайна и работы с современными инструментами фронтенда.  
Проект не претендует на идеальность — это рабочая площадка, где я тренируюсь, пробую новые подходы и постепенно улучшаю свои навыки.

## 🎯 Цели проекта
- Отработка HTML‑структуры и семантики  
- Практика Flexbox и адаптивной сетки  
- Работа с позиционированием и фоновыми изображениями  
- Создание адаптивных секций под разные разрешения  
- Улучшение понимания CSS‑архитектуры  

## 🛠️ Используемые технологии
- **HTML5**
- **CSS3**
- Flexbox / Grid
- Адаптивная вёрстка (медиа‑запросы)
- Работа с кастомными шрифтами

## 📌 Статус проекта
Проект является тренировочным.  
Используется как черновик, экспериментальная площадка и шаг в развитии моих навыков фронтенда.

## 📚 Чему я научился
- Грамотно выстраивать структуру секций  
- Работать с контейнерами и ограничениями ширины  
- Переписывать сложные блоки на Flex  
- Убирать лишние стили и поддерживать чистоту CSS  
- Делать адаптив без хаоса и конфликтов  

## 🚀 Планы на будущее
